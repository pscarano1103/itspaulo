# Design System do Projeto Portfolio

## 1. Introdução

Esta é uma landing page para um portfólio de um desenvolvendor Front End, Com estilo profissional e moderno.

## 2. Fundações

- Pleta de cores:
  --text: #060d08;
  --background: #f6fcf8;
  --primary: #3dd374;
  --secondary: #89efae;
  --accent: #48f486;

- Tipografia
  -Titulos: Montserrat(via Google Fonts)
  -textonormal: Poppins(via Google Fonts)

- Espaçamento
  -Grande (8px ou 16px);

## 3. Componentes

-Botões:
-Todos os botões vão: - Terão uma sombra pequena na parte inferior - Terão arredondamento de 4px - Terão fonte negrito - Seguirão espaçamento definido na fundação - Variações: - Primário: Seguirão a paleta a risca. - Secundário: Terá o valor invertido do fundo/texto

- Campo de Texto (input):
  -Todos os inputs vão:
  - Terão arredondamento de 4px
  - Seguirão espaçamento definido na fundação

## 4. Guia de uso

- No CSS sempre usar variáveis para cores e elementos da fundação ao invés de espalhar pelo código.
